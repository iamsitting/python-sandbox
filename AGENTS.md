# Repository Instructions

- Use `.venv/bin/python` for all Python commands in this repo when available.
- If `.venv` is missing, create it with `python -m venv .venv` and install dependencies from `requirements.txt` before running project commands.
- Prefer running Django commands through the virtual environment, for example `.venv/bin/python manage.py check`.
- Do not assume system Python has the right packages installed.
- Keep changes focused and avoid editing unrelated files.
